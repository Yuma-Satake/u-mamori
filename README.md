## Getting Started

### install pnpm

※もし`pnpm`がインストールされていない場合のみ

```zsh
npm install -g pnpm
```

### install dependencies

```zsh
pnpm i
```

### start dev server

```zsh
pnpm dev
```

### build

```zsh
pnpm build
```

### lint

```zsh
pnpm check
```

## Commands

### GenerateAnalyze

バンドルサイズのマップ（`status.html`）を生成します。

```zsh
pnpm build:analyze
```
